关于在Mac下配置Hexo

关于Hexo在Mac下的配置，百度上可以搜出大量的教程。
但我在配置的过程中出现了一些教程未提及的错误。

##node版本不足##

  • 首先在终端输入执行安装命令

    $ sudo n lastest

    $ sudo hexo s

    $ ls -al [/path/fileName]

    $ sudo chmod 771 [/path/fileName]
    ```

    即可。

##GitHub出现404界面##

  • 此issue隐藏的极为隐蔽，但鲜有教程会提及。

  • 原因是GitHub的Organization和所建立的Repository名称必须保持一致，但可以不区分大小写。
