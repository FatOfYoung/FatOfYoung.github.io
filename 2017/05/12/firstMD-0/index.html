编辑.md文件时的权限问题

  • 有两种解决方案

    1. 提升mac最高root权限
    2. 修改.md文件的权限值

  • 关于第一种可以自行百度,尝试使用第二种方式解决。

    输入

    $ ls -al [/path/fileName]

    发现 .md文件及文件夹针对admin用户组仅提供Read权限。由于我登陆的用户属于admin组别，于是提升组别权限即可。

  • 分别针对.md文件及.md所在文件夹输入命令$ sudo chmod 771 [/path/fileName]

    即可。
