Socat的配置使用

发表于 2017-05-16 | 分类于 Tool | | 阅读次数

小黄书中，狗神在函数打印时使用的是系统自带的Syslogd，
但在iOS9中此功能已失效。于是通过Google找到了一款可替代的插件Socat。

安装

此插件可通过在Cydia搜索SOcket CAT安装。

使用

可分别在Mac和iOS（iPhone需MTerminal插件）中使用。

连接

输入stop可停止接收消息，

exit退出socat

过滤

在多数情况下我们仅需要指定的打印信息，此时可用socat自带过滤功能。
格式为* key value

key值可参照

Key Exp
ASLMessageID asl消息id
Level 进程级别
Time 进程时间
TimeNanoSec 进程时间ns
Sender 进程消息发送者
Facility 进程BundleID
PID 进程id
Message 进程消息内容
ReadUID 读取所使用的用户ID

EXG

最常使用的过滤指令为* Facility [bundleID]
这里以mobike的bundleID为例。

更多指令可参照help
