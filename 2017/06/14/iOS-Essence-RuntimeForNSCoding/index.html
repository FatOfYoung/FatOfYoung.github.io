Runtime与NSKeyedArchiver的应用

发表于 2017-06-14 | 分类于 iOSDe | | 阅读次数

NSKeyedArchiver

NSKeyedArchiver作为轻量级的持久存储方式，使用起来非常的便捷。在实现时，仅需遵守NSCoding协议，加入

- (nullable instancetype)initWithCoder:(NSCoder *)aDecoder
- (void)encodeWithCoder:(NSCoder *)aCoder

两种方法即可。

在原始的方法中，需要根据类的属性分别在以上方法中实现：

  • self.prop = [aDecoder decodeObjectForKey:@"prop"];
  • [aCoder encodeObject:self.prop forKey:@"prop"];

如果属性量比较大，此写法显得略为繁琐。

class_copyIvarList

所有属性名的获取均通过class_copyIvarList保存在ivars中，为了去除类型名前段部分的下划线，增加了substringFromIndex:1方法。

encodeWithCoder

使用SEL替代对属性的getter操作。

initWithCoder

使用 - (void)setValue:(nullable id)value forKey:(NSString *)key;替代对属性的setter操作。
