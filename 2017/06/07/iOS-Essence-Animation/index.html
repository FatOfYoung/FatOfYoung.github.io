UIBezierPath实现的简易动画

发表于 2017-06-07 | 分类于 Animation | | 阅读次数

由于一直对iOS的CoreAnimation富有浓厚的兴趣，在花费5刀拜读了KittenYang的A Guide To iOS Animation后。无意中看到了一款名为Kik的社交软件。

在其中的注册界面出现了检测邮箱的动画。经研究，此动画的关键点在于CoreAnimation之间的协调切换，以及简单三角函数的应用。
于是准备仿造其作为学习Animation之后的练手。

确定CAShapeLayer

CAShapeLayer是CALayer的子类，拥有众多丰富的属性。所以通常使用CAShapeLayer来创建动画及配合Bezier曲线的运用。
因为涉及到三种不同的状态（加载，成功，失败），所以需要创建三个CAShapeLayer便于管理动画的状态。

创建加载动画

首先，根据示例图所示。加载动画需分为四个步骤：

  • 创建CAShapeLayer的stokeEnd属性从0到1由点形成一个闭环的Animation
  • 创建CAShapeLayer的stokeStart属性从0到1由闭环形成一个点的Animation
  • 将以上两步骤添加至AnimationGroup控制其参数
  • 创建Bezier圆形曲线，限定动画的运行轨迹

在尾部的代码中，给Animation进行了setValue操作的目的是便于在CAAnimationDelegate的方法animationDidStop中对各项动画任务进行转换的处理。

由于从圆形的最顶端开始且顺时针。所以startAngle为 - M_PI_2，closewise为YES
